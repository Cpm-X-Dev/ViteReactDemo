## 🎯 Purpose

This repository provides a complete learning experience for building modern web applications with **React**, **TypeScript**, and **Vite**:

- 🌐 A **sample project** — a personal website showcasing real-world React patterns
- 📚 An **interactive learning platform** — 20 guided lessons with live demos and exercises
- ✏️ **Hands-on activities** — scaffolded coding exercises with answer keys
- 🎤 **Presentation slides** — PowerPoint materials used during the workshop
- 📖 **Reference materials** — infographics and visual guides

## 🚀 Getting Started

### Prerequisites

- **Node.js** (v18 or later)
- **npm** (comes with Node.js)

### Quick Start

Each project runs independently. Navigate to the project folder and run:

```bash
# 1. Install dependencies
npm install

# 2. Start the dev server
npm run dev
```

Or use the shortcut command to install and build in one step:

```bash
npm run enroll
```

### Running All Projects

Open three terminals and run each project simultaneously:

```bash
# Terminal 1 — Sample Project (port 3000)
cd ViteSampleProject && npm install && npm run dev

# Terminal 2 — Activities (port 3001)
cd ViteReactActivities && npm install && npm run dev

# Terminal 3 — Learning Platform (port 3002)
cd LearnViteReact && npm install && npm run dev
```
